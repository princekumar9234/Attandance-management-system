# Attendance Management System

A modern, responsive Attendance Management System built with **React** and **Vite**. This application allows teachers or administrators to manage student records and track daily attendance efficiently.

## 🚀 Features

- **Student Management**: Add, edit, and delete student records.
- **Attendance Tracking**: Mark students as Present, Absent, or on Leave.
- **Visual Analytics**: Dynamic charts and progress bars for attendance summaries.
- **Responsive Design**: Premium UI that works seamlessly across mobile, tablet, and desktop.
- **Dark/Light Mode**: User-friendly theme switching.
- **Search & Filter**: Quickly find students by name or roll number.
- **Local Persistence**: Data is automatically saved to LocalStorage.

## 🛠️ Technology Stack

- **Framework**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Modern CSS properties & Glassmorphism)
- **State Management**: React Hooks (useState, useEffect)
- **Icons**: Lucide React / FontAwesome

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/princekumar9234/Attandance-management-system.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd Attendance-System
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
